## To Run The OpenMP Code

```
$ bash openmp.sh [the size of the matrix] [Number of threads]

For example,
Size of matrix = 1000
Number of threads: 4

Run the following:
$ bash openmp.sh 1000 4

The command will generate P, L, A, U files and print the execution time as well as the error magnitude. 
```
